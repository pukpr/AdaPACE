# Gazebo 3D Example

This example demonstrates the Earth-Moon-Sun orbit with an extra Chandler wobble on Earth using the Gazebo 3D rendering app. The main Ada routine governs the orbit and feeds the Gazebo apop 6DOF motions via a shared memory interface defined in the plugins/gazebo directory.
