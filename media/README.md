# media

Images, diagrams, and other media assets used in documentation and examples.

## Conventions

* Architectural diagrams: `media/diagrams/` (SVG preferred; PNG acceptable).
* Screenshots: `media/screenshots/`.
* Use descriptive file names in lower-case with hyphens (e.g. `comms-layer-overview.svg`).
* Do not commit binary assets that can be regenerated; add generation scripts to `docs/` instead.
